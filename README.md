## Группировка строк из БД

Одно из ключевых требований к коду - время
ознакомился с DataFile - там типа телефонные номера
с ними быстрее работать как с целыми числами
в int не поместимся, поэтому long
сначала исходный архив считываем в коллекцию List<Long>
SortSet вроде самый удобный, но очень медленный
HeshSet быстрее, но недостаточно,
будем использовать ArrayList

Для поиска совпадений чисел в определённых колонках - Map
Итоговый список валидных групп - Set
Для записи в итоговый файл - обратно преобразовываем числа в строки
